# Reddit Sentiment Analysis using BERT

This project performs sentiment analysis on Reddit posts using a pre-trained BERT model.

## Objective
To classify Reddit text data into:
- Positive
- Neutral
- Negative sentiments

## Model Used
- BERT (nlptown/bert-base-multilingual-uncased-sentiment)

## Technologies
- Python
- Pandas
- Transformers (Hugging Face)
- Matplotlib

## Features
- Handles real-world text data (emojis, mixed text)
- Automatic text column detection
- Sentiment distribution analysis
- Visualization using graphs
